Regresin lineal sobre slant_up

En este grfico se aplica un modelo de regresin lineal al subconjunto slant_up del conjunto datasaurus_dozen. La lnea roja punteada representa la lnea de tendencia que mejor ajusta los puntos en base a un modelo lineal simple (y = a + bx).

A pesar de que los puntos parecen tener una distribucin con cierta inclinacin visual, la lnea de regresin es casi horizontal. Esto indica que, estadsticamente, la relacin entre x e y es muy dbil o nula desde la perspectiva del modelo lineal. Es decir, la pendiente es muy cercana a cero.

datasaurus_dozen %>%
  filter(dataset == "slant_up") %>%
  ggplot(aes(x = x, y = y)) +
  geom_point(color = "steelblue", size = 2) +
  geom_smooth(method = "lm", se = FALSE, color = "red", linetype = "dashed") +
  theme_minimal() +
  labs(
    title = "Dataset: slant_up con linea de regresion",
    x = "X",
    y = "Y"
  )
